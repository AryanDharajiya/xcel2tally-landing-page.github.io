How it Works?

With Xcel2Tally, It becomes easy to import data of above mentioned templates into tally from excel.

The process is demonstrated in below images.

  1. Create company

    The first step is to create a company, if not already done, in your account for which you want to convert and import your data in tally.

  2. Download sample file

    We have provided sample file for all template type which you can use for your reference while you prepare your excel files.

  3. Prepare your excel files

    You must prepare all your excel files in Xcel2Tally reference format before you upload it into our portal (See second point for details about reference file format).

  4. Select template & upload your file

    Select template type, company, upload prepared excel file and hit process button to start processing.

  5. Download processed file

    Once you submit your file for processing, you can track its progress and download XML file once process is completed.

  6. Import XML file into tally

    Lastly, you just have to import your downloaded XML file into tally.
